# rick_and_morty_widget

## Project structure

### 1. 'client' folder
* App source code

### 2. 'clientDev' folder
* Gulp build folder
* Scss files in 'cliendDev/src/scss'

## How to start
1. git clone https://github.com/GeorgiyZhuravlevRudn/rick_and_morty_widget.git
2. cd 'rick_and_morty_widget/client'
3. npm i
4. npm start
